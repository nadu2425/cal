# Calculator in C

## Description
This is a simple calculator program written in C. It performs basic arithmetic operations such as:

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)

## Requirements
- GCC compiler or any C compiler

## Compilation

Using GCC:

gcc calculator.c -o calculator

## Running the Program

Windows:
calculator.exe

Linux/macOS:
./calculator

## Example

Enter first number: 10
Enter operator (+, -, *, /): +
Enter second number: 5

Result: 15
